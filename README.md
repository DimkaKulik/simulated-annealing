## How to run?

To run you can use 3 following variants:

 - ```console
   $ sudo ldconfig ./lib && ./bin/res.out
   ```
 - ```console
   $ export LD_LIBRARY_PATH=./lib && ./bin/res.out
   ```

 - ```console
   $ make
   ```
